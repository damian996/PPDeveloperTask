Project Description.

The shopping basket web application was written in ASP.NET MVC and AngularJS. 

Server side:

There is no database included in the project for simplicity. 
All sample data is provided by the mock repository that is injected into server side objects by Ninject.
Application uses Web Api controllers to communicate with the client side.

Client side:

Application uses AngularJS framework to communicate with server side api and to control client side MVC views.

NOTE - To reduce the size of the solution, the packages required for the project are not included. 
All required packages should be automatically downloaded by Visual Studio when project is built for the first time. 
If there are any packages that are missing, please download them using NuGet Package Manager.
